# かんと Browser

軽くて、速くて、使いやすいArcライクなブラウザ。

## セットアップ

```bash
# 依存関係をインストール
npm install

# 起動
npm start
```

## 機能

### 垂直サイドバー
- 通常は画面左端に細いラインとして隠れています
- マウスを左端に近づけるとスムーズにスライドして表示
- サイドバー外をクリックすると自動的に隠れます

### タブ管理
- **普通のタブ**: URLナビゲーション
- タブ名・ファビコン・読み込み状態がリアルタイムで更新

### ブックマーク
- **ブクマ1**: サイト状況が保存される
- **ブクマ2**: サイト状況をリセットできる
- 現在のページをワンクリックでブックマーク追加

### キーボードショートカット
| ショートカット | 操作 |
|---|---|
| `⌘T` | 新しいタブ |
| `⌘W` | タブを閉じる |
| `⌘L` | URLバーにフォーカス |
| `⌘R` | ページを再読み込み |
| `⌘[` | 戻る |
| `⌘]` | 進む |
| `⌘1〜9` | タブ切り替え |

## ビルド（配布用）

```bash
npm run build
```
`dist/` フォルダにアプリが生成されます。

## ファイル構成

```
kanto-browser/
├── main.js          # Electronメインプロセス
├── preload.js       # セキュアなIPC bridge
├── package.json
├── assets/
│   ├── icon.png     # アプリアイコン
│   └── icon.icns    # macOS用アイコン
└── src/
    ├── index.html   # ブラウザUI
    └── browser.js   # UI ロジック
```
